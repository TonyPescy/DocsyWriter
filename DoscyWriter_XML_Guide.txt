NOTE: 
	- All comments will be after "//" to denote a comment
	- User entered data will be marked as "UE"
	- Self explanatory tags will be marked with "SE"
	- All strings/text follow this format "<![CDATA[string]]" - Ints dont need it
	- Defualt Time: 0001-01-01
	- Default Date: 00:00:00
	- Handling Bold/Italics/Underline/Strikethrough - keep that order in text for consistency sake
		> Bold: <![CDATA[text[b]boldtext[/b]text]]>
		> Italics: <![CDATA[text[i]italicstext[/i]text]]>
		> Underline: <![CDATA[text[u]underlinetext[/u]text]]>
		> Strikethrough: <![CDATA[text[s]strikedtext[/s]text]]>
		> All: <![CDATA[text[b][i][u][s]affectedtext[/b][/i][/u][/s]text]]>
	- Whole project is within a <YWRITER7> tag as such: <YWRITER7>everything</YWRITER7>
